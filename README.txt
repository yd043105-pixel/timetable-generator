============================================================
  학교 시간표 생성기 (CP-SAT 내장) — 사용설명서
============================================================

화면은 PySide6(Qt) 기반의 밝고 깔끔한 디자인입니다.
OR-Tools CP-SAT 를 내장하여 빈칸이 거의 없는 빡빡한 시간표도
하드 위반 없이 풀어냅니다. 파일 이름은 영문이지만 프로그램 화면과
결과물은 모두 한글입니다.


────────────────────────────────────────────────────────
[A] 처음 한 번만 — EXE 만들기
────────────────────────────────────────────────────────
1) 파이썬이 없다면 https://www.python.org 에서 Python 3.10+ 설치.
   ★ 설치 시 "Add Python to PATH" 를 반드시 체크하세요.

2) 이 폴더의 "build_exe.bat" 를 더블클릭합니다.
   - 라이브러리 설치 -> EXE 생성까지 자동(몇 분 소요).
   - 끝나면 이 폴더 안에 "TimetableGenerator.exe" 가 생깁니다.
     (dist 폴더에도 같은 파일이 있지만, 이 폴더의 것을 쓰면 됩니다.)

3) 이후에는 "TimetableGenerator.exe" 만 더블클릭하면 실행됩니다.
   실행 시 검은 콘솔창은 뜨지 않습니다(프로그램 창만 나옵니다).


────────────────────────────────────────────────────────
[B] 사용 방법
────────────────────────────────────────────────────────
1) [입력/생성] 탭
   (1) 학년도·학기 입력
   (2) "시수표 양식 다운로드" 로 빈 양식을 받아 작성할 수 있습니다.
       또한 "설정 저장(엑셀)" 으로 비수업·교사불가·유사그룹·기본설정을
       엑셀로 보관했다가, 다음에 "설정 불러오기(엑셀)" 로 그대로 적용할 수
       있습니다(프로그램을 껐다 켜도 다시 입력할 필요 없음).

       "시수표(.xlsx) 열기" 로 작성한 시수표를 불러옵니다.
   (3) 비수업 시간: 요일+교시 고른 뒤 [1학년][2학년][3학년][전체학년]
       추가 -> 학년별 칸에 표시됩니다.
   (4) 교사 불가시간: 교사를 고르면 월~금 x 1~7교시 격자가 나옵니다.
       불가능한 칸을 체크하고 [이 교사 불가시간 등록] 을 누르면 한꺼번에
       올라갑니다. 교사를 다시 고르면 그 교사의 기존 설정이 격자에 표시돼
       수정할 수 있습니다.
   (5) 유사과목 그룹: 과목 옆 체크박스를 켜고 -> [그룹 추가]
   (6) 설정: 교사 최대 연속, 하루 시수 여유, 탐색 시간(초)
   (7) (선택) "점심 전후 연속수업 방지" 체크 + 점심 직전 교시 지정
   (8) [시간표 생성] -> 진행 중 현재 페널티가 실시간으로 표시되고,
       언제든 [중단] 으로 멈춰 그때까지의 최선해를 받을 수 있습니다.
   (9) 결과 확인 후:
       - [엑셀로 저장] : 학급별·교사별 시간표(색상 포함)
       - [더 돌리기(이어서)] : 지금 결과에서 점수를 더 낮춤
       - [이어돌리기 저장] : 나중에 다시 불러와 더 돌릴 .json 저장
   (10) 다음에 [이어돌리기 열기] 로 그 .json 을 불러오면,
        시수표 없이도 이전 결과에서 이어서 최적화합니다.

2) [페널티 규칙] 탭 : 지키는 규칙과 최소화하는 규칙 설명.


────────────────────────────────────────────────────────
[C] 팁
────────────────────────────────────────────────────────
- 빡빡한(꽉 찬) 시간표일수록 탐색 시간을 늘리세요(180~300초).
  중간에 충분히 낮아졌다 싶으면 [중단] 으로 멈춰도 됩니다.
- "점심 전후 연속수업 방지" 를 켜면 제약이 강해지니 시간을 더 주세요.
- 점수(페널티)는 낮을수록 좋고, 하드 위반이 0 이어야 쓸 수 있습니다.


────────────────────────────────────────────────────────
[D] 폴더 구성
────────────────────────────────────────────────────────
  gui.py            화면(입력 양식)
  cpsat_solver.py   CP-SAT 솔버(진행상황·중단·이어서돌리기)
  scheduler.py      페널티 모델·후처리
  excel_parser.py   시수표 읽기
  session_io.py     이어돌리기 파일 저장/열기
  models.py         데이터 구조
  output.py         엑셀 저장
  template.xlsx     시수표 양식(프로그램에 내장됨)
  requirements.txt  필요한 라이브러리 목록
  build_exe.bat     EXE 빌드 스크립트
  README.txt        이 파일

────────────────────────────────────────────────────────
[E] 프로그램이 꺼지거나 오류가 날 때
────────────────────────────────────────────────────────
1) 홈 폴더(보통 C:\Users\사용자이름)에 생기는
   "시간표생성기_오류기록.txt" 파일을 열어 내용을 확인하세요.
2) 원인을 더 자세히 보려면 "build_exe_debug.bat" 로 다시 빌드한 뒤
   "TimetableGenerator_debug.exe" 를 실행하세요. 콘솔창이 함께 떠서
   오류 메시지를 직접 볼 수 있습니다.
